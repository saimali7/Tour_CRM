# Tour CRM User Journeys

> Page-by-page flow maps showing screens, states, and user actions

---

## Methodology

Each journey follows this structure:

```
┌─────────────────────────────────────────────────────────────────┐
│  SCREEN NAME                                        [Route]     │
├─────────────────────────────────────────────────────────────────┤
│  ┌─────────────────────────────────────────────────────────┐   │
│  │                    SCREEN LAYOUT                         │   │
│  │  Components and their arrangement                        │   │
│  └─────────────────────────────────────────────────────────┘   │
│                                                                 │
│  ACTIONS:                                                       │
│  • [Button/Link] → Result/Next Screen                          │
│                                                                 │
│  STATES: Loading | Empty | Error | Success                      │
└─────────────────────────────────────────────────────────────────┘
```
